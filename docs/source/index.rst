Overview
===================================

The `Avnet Dual Camera HSIO module <https://avnet.me/dualcam-hsio-module/>`_ is an accessory for the `ZUBoard 1CG <https://avnet.me/zuboard-1cg/>`_ and ships with two IAS camera modules pre-installed. To utilize the out of box image that will demonstrate camera functionality you must mate the camera module board to an ZUBoard 1CG (not included). Use the instructions included in this guide to restore the microSD card to the latest OS image for initial setup or if the microSD card has been erased or reprogrammed.
